Installation Guide

Step-by-step installation via Terminal

  1. ⌘

    Open Terminal

    Press ⌘ + Space to open Spotlight, type Terminal and press Enter.

  2. 2

    Install window-memory

    Paste this command and press Enter. Homebrew will download and install the app — you may be prompted for your Mac password.

    Terminal — Install window-memory
    $ brew install --cask window-memory

    On first launch macOS may ask for Accessibility permission — this is required to read and restore window positions.

  • ✓

    Done

    When the command finishes, open window-memory from Applications or Spotlight (⌘ + Space). Arrange your windows — the app saves the layout automatically.

    • About this app

    window-memory saves your window layout per monitor configuration and restores it automatically when that configuration is detected again.

    No more repositioning windows every time you connect or disconnect an external display.

    Per-monitor layouts

    Saves separately for each combination of displays you use.

    Auto-restore

    Detects display changes and restores positions without any input.

    Menu bar

    Save or restore manually from the menu bar icon at any time.

    No cloud

    Everything stored locally. No account, no tracking, no network.
